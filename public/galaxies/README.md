# Galaxy Textures

Place real galaxy PNG images in this directory. The code auto-loads them
and renders them on the sky sphere using tangent-plane projection.

## Suggested images (all public domain from NASA/ESA Hubble)

Download from https://esahubble.org/images/ or https://www.nasa.gov/

| File            | Object     | Type           | Search term                     |
|-----------------|------------|----------------|---------------------------------|
| m31.png         | Andromeda  | Large spiral   | "Hubble Andromeda"              |
| m51.png         | Whirlpool  | Spiral         | "Hubble Whirlpool Galaxy"       |
| m104.png        | Sombrero   | Edge-on        | "Hubble Sombrero Galaxy"        |
| m82.png         | Cigar      | Starburst      | "Hubble M82 Cigar Galaxy"       |
| ngc1300.png     | NGC 1300   | Barred spiral  | "Hubble NGC 1300"              |
| m87.png         | M87        | Elliptical     | "Hubble M87 galaxy"            |
| ngc4565.png     | NGC 4565   | Edge-on spiral | "Hubble NGC 4565 Needle Galaxy"|

## Format
- PNG or JPG
- Any resolution (auto-resized to 256x256)
- Square crop works best
- Black background is fine (auto-treated as transparent)
